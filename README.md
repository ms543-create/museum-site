## Design Framework

### Design Style: Swiss
The site uses a Swiss design style with clean layouts, strong typography, and minimal visual clutter. This helps present information clearly and keeps the focus on content.

### Cialdini Principle: Authority
The site uses authority by presenting information in an educational, research-based format. The structured layout and exhibit-style design make the content feel credible and trustworthy.

### Brand Archetype: The Sage
The site follows the Sage archetype, focusing on knowledge, learning, and explanation. The tone is educational and analytical, helping users understand how consumer experiences evolved.
